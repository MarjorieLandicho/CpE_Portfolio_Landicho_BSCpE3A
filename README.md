Description: Assembled and calibrated a custom, open-sided cantilever 3D printer featuring an unsupported X-axis printing arm.  4. Digital Logic Circuit Prototype
Platform/Tools: Solderless Breadboard, Integrated Circuits
 Description: Realized and tested a physical hardware subsystem using a full-sized solderless breadboard. The system utilizes three dual in-line package (DIP) IC chips mounted across the central isolation divider to route logic signals. It features an optimized power distribution network alongside discrete components like a signal diode and output LEDs for physical hardware-in-the-loop state debugging.
Key Deliverables: Mapped pinout wiring network, balanced power bus distribution, and hardware state verification.
